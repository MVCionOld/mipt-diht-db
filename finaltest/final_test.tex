\section*{Задание 2: запросы}
Получить все уникальные имена спортсменов из Ваканды, родившихся не позднее 11 сентября 2001 года:
\newline
\begin{enumerate}
    \item 
        \[S_0 = (SPORTSMAN[country\_nm = "\textup{Wakanda}" \land \\* birth\_dt \leq "\textup{2001-09-11}"]);\]
        \[S_1 = S_0[sportsman\_id, full\_mn], \hspace{1cm} S_2 = S_1;\]
        \[D = S_1[S_1.sportsman\_id \neq S_2.sportsman\_id \land \\* S_1.full\_nm = S_2.full\_nm]S_2;\]
        Ответ: \[U = (S_1 - D)[\#.full\_nm]\]
    \item 
        \[S_0 = \{s: P_{SPORTSMAN}s \land \\* s[country\_name] = "\textup{Wakanda}" \land s[birth\_dt] \leq "\textup{2001-09-11}"\};\]
        \[S_1 = \{s[sportsman\_id, full\_mn]: P_{S_0}s\}, \hspace{1cm} S_2 = S_1; \]
        Ответ: \[U = \{s[full\_nm]: P_{S_1}s \land \neg\exists P_{S_2}p (s[sportsman\_id] \neq p[sportsman\_id] \land s[full\_nm] = p[full\_nm])\}.\]
\end{enumerate}

\section*{Задание 5: sql-запрос}
Получить всех спортсменов из Ваканды, а также дату их рождения, хоть раз установивших мировой рекорд на соревнованиях в Бангладеше:
\newline
\begin{verbatim}
WITH 
  WR_HOLDER AS (
    SELECT DISTINCT
      RESULT.sportsman_id
    FROM
      RESULT
    INNER JOIN
      COMPETITION
      ON
        RESULT.competition_id = COMPETITION.competition_id
    WHERE 1 = 1
      AND COMPETITION.competition_place_txt = 'Bangladesh'
      AND RESULT.world_record_flg
  )
SELECT DISTINCT
  SPORTSMAN.full_nm
  , SPORTSMAN.birth_dt
FROM
  SPORTSMAN
WHERE 1 = 1
  AND SPORTSMAN.sportsman_id IN (
    SELECT
      WR_HOLDER.sportsman_id
    FROM
      WR_HOLDER
  )
  AND SPORTSMAN.country_nm = 'Wakanda';
\end{verbatim}

\section*{Задание 6: sql-запрос}
Получить наиболее успешных спортсменов с максимальным рекордом, сгруппированных по стране и отсортированных по стране:
\newline
\begin{verbatim}
WITH
  BEST_RESULT AS (
    SELECT
      max(SPORTSMAN.personal_record_value) AS personal_record_value
      , SPORTSMAN.country_nm
      , SPORTSMAN.sport_type_nm
    FROM
      SPORTSMAN
    GROUP BY
      SPORTSMAN.country_nm
      , SPORTSMAN.sport_type_nm
  )
SELECT
  S.full_nm
  , S.birth_dt
  , S.country_nm
  , S.sport_type_nm
  , S.personal_record_value
FROM
  SPORTSMAN AS S
WHERE
  S.personal_record_value = (
    SELECT
      max(BEST_RESULT.personal_record_value)
    FROM
      BEST_RESULT
    WHERE 1 = 1
      AND BEST_RESULT.sport_type_nm = S.sport_type_nm
      AND BEST_RESULT.country_nm = S.country_nm
  )
ORDER BY
  S.country_nm;
\end{verbatim}
